Class DataChunk

DataSource that represents a section (chunk) of a larger data source (parent). Useful when chunking a data source to parallelize processing.

    • Defined in src/chunk/DataChunk.ts:70

    NOTE: If this chunk belongs to a remote DataSource, this function waits until the data for this chunk has been transferred from the remote and into memory. Also, if the final size for the remote DataSource is unknown, the byteLength of this chunk could change after it finishes loading.
